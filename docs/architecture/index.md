## Runtime topology (one-line)

```
Browser (Nuxt SPA) ── $fetch + EventSource ──► Play :9000 ── LlmProvider ──► OpenAI/OpenRouter/Ollama
                                                        ├── TaskPollerJob @Every(30s) on virtual threads
                                                        ├── Channel adapters (Slack / Telegram / WhatsApp)
                                                        └── JPA → H2 (dev) / PostgreSQL (prod template)
```

## Conventions cheat sheet

- **Play 1.x:** static-method controllers/services/jobs. No DI container.
- **Auth:** cookie session; `AuthCheck @Before`; webhooks exempt. Unauthenticated = **HTTP 401**; genuine authorization failures = **403**.
- **Transactions:** `services.Tx.run(...)` wraps `JPA.withTransaction`. Re-fetch entities after nested `Tx.run`.
- **JSON:** single `utils.GsonHolder.INSTANCE`.
- **Enums in DB:** string-backed, manual conversion (hot-reload constraint).
- **Full-text search:** none today (LIKE + regex). Future: Postgres `tsvector`, not H2 FT.
- **Test commands:** `play auto-test` (NOT `play test`) + `cd frontend && pnpm test` after every frontend edit.
- **Commit flow:** bump `application.version` → push to **both** remotes (`origin` Bitbucket + `github`).
