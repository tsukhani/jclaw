## Description: <br>
Produces polished HTML design artifacts such as slide decks, interactive prototypes, hi-fi mockups, animated videos, landing pages, dashboards, and marketing one-pagers. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[cced3000](https://clawhub.ai/user/cced3000) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Designers, product teams, marketers, and developers use this skill to turn visual design requests into reviewable HTML/CSS/JS/SVG artifacts, including decks, prototypes, mockups, dashboards, videos, landing pages, and design variations. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The agent may read design-related project files and write prototype, deck, or mockup assets. <br>
Mitigation: Install and run the skill only in workspaces where that file access and asset creation are acceptable. <br>
Risk: Generated HTML may load public CDNs or use host-provided LLM helpers. <br>
Mitigation: Review generated HTML and external resource references before sharing or deploying artifacts. <br>
Risk: Marketplace capability tags may overstate support for crypto, purchases, or sensitive credentials. <br>
Mitigation: Correct or ignore those tags during governance review because the security evidence does not support those capability claims. <br>
